Setup

Clone and make sure you can build the Godot repository by following the instructions here : https://docs.godotengine.org/en/stable/development/compiling/index.html#

In the godot/modules folder clone the Psionic-Godot repository with the following command : git clone --recurse-submodules https://github.com/KerDelos/Psionic-Godot.git

Rename the folder from Psionic-Godot to gdpsionic

Compile Godot & enjoy
